## 📖 Sobre el Proyecto

Esta plataforma web está diseñada para brindar a los pacientes del **Centro Médico Cardiología y Pediatría** un acceso rápido y sencillo a la información del consultorio. Permite conocer los servicios disponibles, el equipo médico y facilita la reserva de **turnos de forma 100% online**.

El centro cuenta con atención especializada en dos ramas clave de la medicina:
- **Cardiología Clínica:** A cargo de la **Dra. Avila María del Carmen**.
- **Pediatría Integral:** A cargo de la **Dra. Avila Mirta Susana**.

<br />

## ✨ Características Principales

- 📅 **Reserva de Turnos Online:** Integración directa con plataforma de turnos para agilizar las citas médicas.
- 🫀 **Estudios Cardiológicos:** Información y disponibilidad de estudios como **Holter 24hs**, **MAPA** y **Electrocardiograma (ECG)**.
- 📱 **Diseño Responsivo:** Experiencia de usuario optimizada para que los pacientes puedan navegar desde sus teléfonos móviles, tablets o computadoras de escritorio.
- 📍 **Información de Contacto y Ubicación:** Datos claros y mapa interactivo para facilitar el arribo al consultorio.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue desarrollado utilizando herramientas modernas para garantizar la mejor performance y escalabilidad:

- **[React](https://reactjs.org/)** (v18)
- **[Vite](https://vitejs.dev/)** (Empaquetador y servidor de desarrollo ultrarrápido)
- **HTML5 & CSS3**
- Buenas prácticas de **SEO** e implementación de **Schema.org** para mejorar el posicionamiento orgánico con etiquetas de negocio médico.

## ⚙️ Instalación y Desarrollo Local

Si deseas clonar y correr el proyecto en tu entorno local, sigue estos simples pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/drasavila-consultorios.git
   ```

2. **Navegar a la carpeta del proyecto:**
   ```bash
   cd drasavila-consultorios
   ```

3. **Instalar las dependencias:**
   ```bash
   npm install
   ```

4. **Levantar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```
